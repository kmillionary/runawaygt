# Runaway GT

Landing page estatica para tours y ascensos a volcanes de Guatemala.

## Publicacion

El proyecto esta preparado para desplegarse en GitHub Pages mediante GitHub Actions.

## Estructura

- `index.html`: landing principal
- `lugar.html`: pagina de detalle por destino
- `styles.css`: estilos globales
- `app.js`: contenido y render dinamico
- `images/`: assets visuales
- `vendor/fontawesome/`: Font Awesome local
